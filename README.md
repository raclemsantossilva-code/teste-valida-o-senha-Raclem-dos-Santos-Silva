# Teste Validação de Senha

Projeto desenvolvido para o desafio técnico de validação de senha.

## Regras de Validação

Uma senha é considerada válida se:

- Possuir **9 ou mais caracteres**
- Conter **ao menos 1 dígito**
- Conter **ao menos 1 letra minúscula**
- Conter **ao menos 1 letra maiúscula**
- Conter **ao menos 1 caractere especial**

Caracteres especiais permitidos:

@ # $ % ^ & * ( ) - +

- **Não pode conter caracteres repetidos**
